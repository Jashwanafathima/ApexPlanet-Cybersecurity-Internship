# 🌐 Networking Basics

Networking is the backbone of cybersecurity. It helps understand how systems communicate and how attacks occur.

---

## 📡 OSI Model (7 Layers)

The OSI model defines how data travels through a network.

1. **Application** → User interaction (HTTP, FTP)
2. **Presentation** → Data encryption & formatting
3. **Session** → Session management
4. **Transport** → Data transfer (TCP/UDP)
5. **Network** → Routing (IP addresses)
6. **Data Link** → MAC address communication
7. **Physical** → Hardware (cables, signals)

## 🌍 TCP/IP Model (4 Layers)

Simplified version of OSI:

1. Application
2. Transport
3. Internet
4. Network Access

---

## 🌐 DNS (Domain Name System)

* Converts domain names into IP addresses

Example:
google.com → 142.250.x.x

---

## 🌐 HTTP vs HTTPS

| HTTP          | HTTPS     |
| ------------- | --------- |
| Not secure    | Secure    |
| No encryption | Encrypted |
| Port 80       | Port 443  |

---

## 🌐 IP Address

An IP address uniquely identifies a device in a network.

Example:
192.168.56.101

---

## 🌐 Types of IP

* Private IP → Used inside local network
* Public IP → Used on the internet

---

## 🌐 NAT (Network Address Translation)

* Converts private IP to public IP
* Helps devices access the internet

---

## 🌐 Ports

Ports identify specific services.

Examples:

* 21 → FTP
* 22 → SSH
* 80 → HTTP
* 443 → HTTPS

---

## 🌐 Protocols

* TCP → Reliable communication
* UDP → Faster but less reliable

---

## 🎯 Importance in Cybersecurity

* Helps understand network attacks
* Used in scanning and monitoring
* Essential for tools like Nmap and Wireshark
