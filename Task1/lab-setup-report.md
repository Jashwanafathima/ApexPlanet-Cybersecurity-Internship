### Lab Setup Report

**Objective:**
To set up a safe and isolated cybersecurity lab environment for testing and learning purposes.

**Tools Used:**

* Kali Linux (Attacker Machine)
* Metasploitable2 (Target Machine)
* Oracle VirtualBox

**Steps Performed:**

1. Installed Oracle VirtualBox.
2. Imported Kali Linux virtual machine.
3. Installed Metasploitable2 as the target system.
4. Configured both machines to use a Host-Only Adapter network.
5. Verified connectivity between machines using the ping command.

**Result:**
Successfully established communication between Kali Linux and Metasploitable2 in a private lab network.

**Learning Outcome:**
Learned how to create a virtual cybersecurity lab and configure network settings for safe testing.
